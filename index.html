The conversion moves the 72-byte block that sits at offset 75440 (MH4U) / 80296 (MH4G) of the decrypted save, exactly like the reference QuickBMS + Python tool. The checksum is recalculated and the file is re-encrypted with its original key.

The extra data files can be copied over from a game to another without conversion, but only the card 1, card 2, card 3 and cardbox. The other files should be replaced with their respective region versions since they contain DLC compatible with the game.
